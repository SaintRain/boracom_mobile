БОРА

Главная

ГК «БОРА» – крупный российский производитель современного автоматизированного оборудования и линий для обработки тонколистового металла.

Мы производим профилегибочное оборудование для производства металлопрофиля, строительного кровельного и промышленного профиля.

Вся наша продукция – оборудование, обладающее большим рабочим ресурсом, имеющее высокие показатели производительности и рассчитанное на интенсивную эксплуатацию.

Компания «Бора» – одно из ведущих предприятий отрасли. Созданная в 2001 году на производственной базе 1-го Московского приборостроительного завода, компания в настоящее время выпускает более 50 единиц автоматизированных профилегибочных линий ежегодно.

Профилегибочное оборудование

  • Линии профнастила – для изготовления всех типов профнастила (от С8 до Н 158, с высотой гофры до 158 мм)
  • Линии металлочерепицы – для производства металлочерепицы различных профилей и декоративных форм
  • Линии сайдинга – для выпуска самых востребованных профилей металлосайдинга
  • Линии резки металла – для раскроя рулонного листового металла (поперечный, продольный, продольно-поперечный способы раскроя)
  • Оборудование для производства фасадных панелей
  • Кровельный фальцепрокатный станок – для производства фальцевой кровли
  • Размотчики – оборудование различной грузоподъемности для размотки рулонного металла
  • Оборудование для производства лстк - автоматизированные комплексы
  • Универсальный прокатный стан для производства профилей строительного назначения
  • Автоматизированные линии для производства металлического штакетника различного сечения
  • Линии для производства доборных элементов кровли
  • Оборудование для производства металлического (шляпного) профиля обрешётки
  • Строительные профили различного назначения
  • Вспомогательное оборудование

автоматизация производства

Автоматизированная система учета металла, контроля и распределения производственных заданий для прокатного производства.

Производство

Производство компании ГК БОРА, это токарные, фрезерные, раскройные обрабатывающие центры с ЧПУ, парк универсального оборудования. Производственный процесс изготовления профилегибочных валков основан на без чертежном методе производства, что значительно ускоряет время изготовления комплектов валков и их доводку при наладке новых изделий.

Наша компания располагает собственным конструкторским бюро, которое ведёт разработки с использованием современных программ CAD/CAM.

Ассортимент выпускаемой продукции достаточно широк. От мобильных фальцепрокатных станков до автоматизированных профилегибочных комплексов со штамповкой и перфорацией профиля.
